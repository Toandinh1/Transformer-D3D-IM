# TransD3D-IM
This repo is the official implementation of **[Transformer-Based Deep Learning Detector for Dual-Mode Index Modulation 3D-OFDM](https://ieeexplore.ieee.org/abstract/document/10317107)** paper. This repository allows researchers and practitioners to reproduce our results as well as a baseline.
